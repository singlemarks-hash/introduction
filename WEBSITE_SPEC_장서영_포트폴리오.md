## 0. 프로젝트 개요

- **목적**: 방송·미디어 관계자에게 보내는 장서영(인물) + 작은따옴표(공간) 소개 포트폴리오 웹사이트
- **권장 스택**: 정적 사이트 (HTML/CSS/JS 단일 페이지 또는 Astro/Next.js). 외부 CMS 불필요
- **페이지 구성**: ① 메인(인물) ② 공간(작은따옴표) — 상단 네비게이션으로 전환. 원페이지 스크롤 + 앵커도 허용
- **반응형 필수**: 모바일에서 표는 세로 스택으로 변환

## 1. 디자인 시스템 (docx와 동일하게)

```css
:root {
  --accent:   #8B6F47;  /* 웜 브라운 — 제목, 카드 타이틀, 강조 */
  --dark:     #2B2B2B;  /* 본문 */
  --gray:     #666666;  /* 보조 텍스트 */
  --bg-card:  #F5F1E8;  /* 크림 — 카드/라벨 배경 */
  --bg-photo: #FAF7F0;  /* 사진 placeholder 배경 */
  --white:    #FFFFFF;
}
```

- **폰트**: 웹에서는 Pretendard(1순위) 또는 Noto Sans KR — docx의 맑은 고딕과 같은 고딕 계열 유지
- **UI 원칙 (중요)**: 서술형 장문 금지. **표·카드 중심의 스캔 가능한 레이아웃** ("한국 커뮤니티기업 느낌"). 짧은 명사형 카피
- **표 스타일**: 라벨 셀 = `--bg-card` 배경 + 볼드 / 3열 표 헤더 = `--accent` 배경 + 흰 글씨
- **카드 행 규칙 (Fit)**: [내용 | 사진] 2열 행. 행 높이 통일. 가로 사진은 사진칸 폭 100% Fit, 세로로 긴 이미지는 통일 행높이에 맞춰 축소 후 중앙 배치. 사진칸 여백 0
- **인용문구 박스 사용 금지** (확정된 디자인 결정)
